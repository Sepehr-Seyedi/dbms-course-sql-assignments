Assignment 1

Collection of SQL assignments from my university DBMS course, including table creation, data insertion, and various query exercises using Suppliers–Parts–Shipments schema.
1. Find the names of all suppliers who supply part P2 using a subquery with IN.

2. Retrieve the names of suppliers who supply part P2 using the EXISTS operator.

3. Find the names of suppliers who supply any part that has the color 'RED'.

4. Find all parts that are supplied by more than one supplier.

5. For each part, find the maximum quantity supplied.

6. Combine the part numbers of parts weighing more than 16 with the part numbers supplied by supplier S2.
